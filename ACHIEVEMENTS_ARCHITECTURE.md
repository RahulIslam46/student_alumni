## Security Architecture

```
┌─────────────────────────────────────────────────────────────┐
│                      Security Layers                         │
├─────────────────────────────────────────────────────────────┤
│  Input Validation                                            │
│  ┌────────────────────────────────────────────────────────┐ │
│  │ • File type checking (JPG, PNG, GIF)                   │ │
│  │ • Text field validation (not empty)                    │ │
│  │ • ID format validation (numeric)                       │ │
│  └────────────────────────────────────────────────────────┘ │
├─────────────────────────────────────────────────────────────┤
│  SQL Injection Prevention                                    │
│  ┌────────────────────────────────────────────────────────┐ │
│  │ • Prepared Statements (all queries)                    │ │
│  │ • Parameter binding (no string concatenation)          │ │
│  │ • Parameterized queries                                │ │
│  └────────────────────────────────────────────────────────┘ │
├─────────────────────────────────────────────────────────────┤
│  File System Security                                        │
│  ┌────────────────────────────────────────────────────────┐ │
│  │ • Timestamp-based naming (prevents overwrite)          │ │
│  │ • Dedicated directory (isolated storage)               │ │
│  │ • Path validation                                      │ │
│  └────────────────────────────────────────────────────────┘ │
├─────────────────────────────────────────────────────────────┤
│  Error Handling                                              │
│  ┌────────────────────────────────────────────────────────┐ │
│  │ • Try-catch blocks (all DB operations)                 │ │
│  │ • User-friendly error messages                         │ │
│  │ • Logging (SQLException details)                       │ │
│  └────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────┘
```
